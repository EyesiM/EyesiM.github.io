Ubuntu 14.10 Mongodb 安装

2016/10/02 • 分类: Mongodb

新版的 Mongodb 不支持 Ubuntu 的 14.10 版本，这个系统的版本挺尴尬的。
所以去用了旧版的 Mongodb –> 2.6。

Mongodb ubuntu 的安装指南官网链接：
https://docs.mongodb.com/v2.6/tutorial/install-mongodb-on-ubuntu/

安装步骤，个人经验：

  1. sudo apt-key adv –keyserver hkp://keyserver.ubuntu.com:80 –recv 7F0CEB10
  2. echo ‘deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen’ | sudo tee /etc/apt/sources.list.d/mongodb.list
  3. sudo apt-get install -y mongodb-org=2.6.9 mongodb-org-server=2.6.9 mongodb-org-shell=2.6.9 mongodb-org-mongos=2.6.9 mongodb-org-tools=2.6.9
    （步骤三这里，官网先是更新库再去下载，由于 ubuntu 14.10 比较特别，就直接指定版本下载了）
  4. 安装完后
    用命令开启mongodb：
    • sudo service mongod start
    • mongod
      开启可能会遇到各种错误报告，例如 dbpath:/data/db not exits，port was in use 等等，具体的解决方法主要参照官网
      https://docs.mongodb.com/manual/tutorial/manage-mongodb-processes/
