关于 border-radius 自己要注意的几点

2016-07-14 • 分类: CSS

关于 border-radius
Safari 浏览器、Firefox 浏览器、Opera 浏览器及 Chrome 浏览器都支持这种绘制圆角边框的样式。

但是针对不同浏览器的写法要注意:
要注意添加上浏览器前缀;

- Firefox 为'-moz-border-radius'的形式
- Opera 为 'border-radius'的形式
- Chrome 为 'border-radius' 或 '-webkit-border-radius' 的形式。

在 border-radius 属性中，可以指定两个半径。
在 Chrome 浏览器与 Safari 浏览器中，会绘制出一个椭圆形边框，一个半径为椭圆的水平方向半径，第二个半径为椭圆的垂直方向半径。
但是在 Firefox 浏览器与 Opera 浏览器中，将第一个半径作为边框左上角与右下角的圆半径来绘制，将第二个半径作为边框右上角与左下角的圆半径来绘制。

boder 设置为 none ，并不影响 border-radius 的显示
